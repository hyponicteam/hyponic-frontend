# Getting Started with Hyponic App

Hyponic adalah aplikasi panduan petani hidroponik

## Cara menjalankan Aplikasi

In the project directory, you can run:

### install nodemodules

<code>npm install</code>

### Jalankan Aplikasi

<code>npm start</code>
